Урок №4. Свойства CSS для работы со шрифтами

Подключение файлов шрифтов на CSS странице

Запись в CSS файле

@font-face {
font-family: 'Gilroy';
src: url('../fonts/Gilroy-BoldItalic.eot');
src: local('Gilroy Bold Italic'), local('Gilroy-BoldItalic'),
url('../fonts/Gilroy-BoldItalic.eot?#iefix') format('embedded-opentype'),
url('../fonts/Gilroy-BoldItalic.woff') format('woff'),
url('../fonts/Gilroy-BoldItalic.ttf') format('truetype');
font-weight: bold;
font-style: italic;
}

Результат

Я заголовок второго уровня, выведенный шрифтом Gilroy, который подключен с папки fonts.

Свойство font-family

Если название шрифта состоит из 2 слов, он заключается в кавычки.

emmet - ff, ffa (font-family: arial)

Значения свойства font-family
serif - шрифт с засечками
sans-serif - рубленный шрифт (без засечек)
cursive - курсивные шрифты
fantasy - декоративные шрифты
monospace - моноширинные шрифты

Запись в CSS файле

p.arial{
font-family: Arial, "Helvetica Neue", Helvetica, sans-serif;
font-size: 18px;
font-weight: bold;
}

p.gilroy{
font-family: Gilroy;
font-size: 18px;
}

.proximanova{
font-family: ProximaNova;
font-style: italic;
font-size: 18px;
}

Результат

Я параграф выведенный шрифтом Arial. Я жирный и размер у меня 18 пикселей.

Я параграф выведенный шрифтом Gilroy. У меня размер тоже 18 пикселей

Я параграф выведенный шрифтом ProximaNova, стиль у меня курсивный. Размер так же у меня 18 пикселей.

Свойство font-size.

Font-Size - размер шрифта. По умолчанию в браузере используется размер 16px.

emmet - fz16 (font-size: 16px)

Запись в CSS файле

.size16{
font-size: 16px;
}

.size18{
font-size: 18px;
}

.size24{
font-size: 24px;
}

Результат

Я параграф. У меня размер шрифта 16 пикселей

Я параграф. У меня размер шрифта 18 пикселей

Я параграф. У меня размер шрифта 24 пикселя

Свойство Font-Style

Значения свойства Font-Style
font-style: normal (по умолчанию)
font-style: italic (система подключает отдельный пакет шрифтов, а не просто наклоняет текст)
font-style: oblique (как раз наклоняет шрифт)

emmet - fsi (font-stile: italic)

Запись в CSS файле

.normal{
font-style: normal;
}

.italic{
font-style: italic;
}

.oblique{
font-style: oblique;
}

Результат

Я параграф, написанный со стилем normal. Мой стиль используется по умолчанию браузером.

Я параграф, написанный со стилем italic. Для моего написания, система подключает отдельный пакет шрифтов, а не просто наклоняет текст.

Я параграф, написанный как раз наклоненным стилем, только визуально не понятна разница между моим стилем и стилем italic

Свойство Font-Weight

Свойство Font-Weight определяет толщину (жирность) шрифта и имеет следующие значения:

font-weight
100 - thin
300 - lite
400 - normal
500 - medium
600 - semibold
700 - bold
900 - black

Если указать свойство, которое не существует у этого шрифта, то свойство не применится и будет отображаться normal

emmet - fw700 (font-weight: 700)

Запись в CSS файле

.fw-thin{
font-weight: 100;
}

.fw-lite{
font-weight: 300;
}

.fw-normal{
font-weight: normal;
}

.fw-medium{
font-weight: 500;
}

.fw-semibold{
font-weight: 600;
}

.fw-bold{
font-weight: 700;
}

.fw-black{
font-weight: 900;
}

Свойство Color

Свойство color - определяет цвет текста. Это свойство наследуется для всех на странице, кроме ссылок.

emmet - с (color:#000)

Типы записи значений color:
red - редко используется на практике;
#ffffff - шестнадцатеричный код цвета (часто применяется)
rgba(59, 180, 59, 0.4); - (rgba код цвета с прозрачностью)

Запись в CSS файле

.green{
color:green;
}

Свойство text-align определяет горизонтальное выравнивание текста в элементе.

Значения свойства Text-Align
left (по умолчанию)
right (справа)
center (по центру)
justify (распределяет по всему элементу)

emmet - tac (text-align: center)

Запись в CSS файле

.left{
text-align-last: left;
}

.right{
text-align: right;
}

.center{
text-align: center;
}

.justify{
text-align: justify;
}

Свойство text-decoration

Свойство text-decoration определяет оформление текста.

emmet - tdu (text-decoration: underline)

Значения свойства Text-Decoration
none (без декорации) (по умолчанию)
underline (линия под текстом)
overline (линия над текстом)
line-through (зачеркнутый)

Зпись в CSS файле

.underline{
text-decoration: underline;
}

.overline{
text-decoration: overline;
}

.through{
text-decoration: line-through;
}

Свойство Text-Shadow

Свойство Text-Shadow - определяет тень под текстом.

Формат записи:

Запись в CSS файле

.shadow{
text-shadow: 2px 2px 3px rgba(0, 0, 0, 0.8)
}

Свойство Text-Transform

Свойство text-transform контролирует в тексте выделение заглавными буквами

Значения свойства Text-Transform
none - (по умолчанию)
uppercase - (текст в верхнем регистре)
lowercase - (текст в нижнем регистре)
capitalize - (каждое слово с заглавной буквы)

emmet - ttu (text-transform: uppercase)

Запись в CSS файле

.uppercase{
text-transform: uppercase;
}

.lowercase{
text-transform: lowercase;
}

.capitalize{
text-transform: capitalize;
}

Свойство Line-Height

Свойство Line-Height - определяет высоту строки. Делает текст более разреженным. Равна высоте шрифта (по умолчанию). Можно записывать процентами, пикселями, единицами.

Запись в CSS файле

.height{ line-height: 200%; }
